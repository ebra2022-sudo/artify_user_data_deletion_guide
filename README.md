
# Artify – Account Deletion Instructions

If you wish to delete your Artifuy account, please follow these steps:

## Step 1: Contact Support
Send an email to: **support@artifuy.com**  
Please use the same email you registered your Artifuy account with.

## Step 2: What We Delete
- Your Firebase Authentication account
- All associated user data stored in our system

## Step 3: Timeframe
Account deletion requests are processed within **24–48 hours**.
